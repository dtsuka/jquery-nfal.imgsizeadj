NFAL Image Size Adjuster - jQuery Plugin

画像を親要素に合わせて縮小し、中心に配置するプラグインです

  • Example
  • Options
  • Download

Example

オリジナル画像（600×450px）

1.親要素を埋めるように配置

$('.adjust1').imgSizeAdj({mode:'outer'});

2.親要素からはみ出ないように配置

$('.adjust2').imgSizeAdj({mode:'inner'});

Options

Name デフォルト値 説明
mode 'outer' 'outer' 親要素を埋めるように配置
'inner' 親要素からはみ出ないように配置
parent null 親要素を指定する場合
例) parent:'.parent'
horizontalCentering true 水平方向のセンタリングを行わない場合はfalse
verticalCentring true 垂直方向のセンタリングを行わない場合はfalse
